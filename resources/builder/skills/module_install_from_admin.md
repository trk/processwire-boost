# Install module from Admin
description: Install a site module via Admin (Modules > New).
## Blueprints
- .ai/blueprints/pw_core/Modules.json
## Steps
- Go to Modules > New
- Choose: Directory, Upload ZIP, or enter ZIP URL
- Click Install
## Request
install module from ZIP URL
## Response
module installed and listed under Site tab
## Example
```text
Admin → Modules → New → “Download from URL” → paste https://.../Module.zip → Install
```
